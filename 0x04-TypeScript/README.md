
# Frontend JavaScript

This repository contains TypeScript projects and exercises.

## Project Structure

- `0x04-TypeScript/` - TypeScript tasks and exercises

## Requirements

- Node.js
- npm or yarn
- TypeScript

## Setup

Navigate to the specific task directory and install dependencies:

```bash
cd 0x04-TypeScript/task_0
npm install
```

## Build

To compile the TypeScript code:

```bash
npm run build
```

## Development

To start the development server:

```bash
npm run start-dev
```
